# 学习系统设计

## 目标

把 AI Agent 学习过程拆成四层：

1. 捕获：快速保存材料、想法和问题。
2. 理解：把原始材料整理成自己的解释。
3. 实践：用代码、项目和真实任务验证理解。
4. 沉淀：形成可复用的方法论、模板和判断标准。

## 工具边界

- Obsidian 负责阅读、浏览、双链和回顾。
- GitHub 负责长期同步和版本历史。
- Codex 负责批量整理、改写、生成索引、抽取结构和辅助代码实验。
