# 学习与认知迭代总览

这个仓库同时服务两条主线：

1. 学习：课程、论文、概念、代码实验、项目经验。
2. 认知迭代：DeepMe 笔记、每日反思、判断变化、方法论更新。

## 日常入口

- `00-inbox/`：先捕获，不打断学习和思考。
- `60-cognition/`：记录判断变化和每日反思。
- `01-maps/AI Agent与机器学习学习地图.md`：AI Agent 和机器学习学习路线。
- `50-systems/长期学习工作流.md`：长期学习与认知迭代流程。

## 每晚反思问题

1. 今天我有哪些判断发生了变化？
2. 这个变化来自事实、体验、他人反馈，还是情绪？
3. 哪个变化值得写进 `60-cognition/`？
4. 哪个变化只是临时感受，暂时不要沉淀？
5. 明天应该验证什么？

## DeepMe 同步

DeepMe 是认知迭代来源之一。同步后建议放到：

```text
60-cognition/deepme/
```

同步后的内容不要直接当作稳定结论。先保留来源，再由每日反思或周复盘提炼出真正改变判断的部分。
