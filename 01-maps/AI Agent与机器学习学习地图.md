# AI Agent 与机器学习学习地图

## 当前主线

目标不是收集资料，而是形成能解释、能实现、能判断取舍的知识体系。

```text
机器学习基础 -> 深度学习 -> LLM -> RAG / Tool Use -> Agent 系统 -> 产品与部署
```

## 学习闭环

每个主题都按同一条路径推进：

1. 来源笔记：这门课、这场讲座、这篇论文到底讲了什么。
2. 概念卡片：我能不能脱离原文讲清楚一个概念。
3. 自测问题：我能不能从记忆中回答关键问题。
4. 代码实验：我能不能用代码或推导验证。
5. 应用判断：这个概念在真实项目里有什么用，什么时候不用。
